# قشطة

## Vision

قشطة هي مجموعة ألعاب ويب قصيرة للعائلة.

كل لعبة يجب أن تكون ممتعة، بسيطة، وسهلة المشاركة.

الهدف ليس إطالة وقت اللعب، بل خلق لحظات جميلة بين أفراد الأسرة.

---

# Mission

بناء أفضل مجموعة ألعاب عربية قصيرة للعائلة.

اللعبة يجب أن تكون ممتعة للأطفال ومطمئنة للوالدين.

---

# Target Audience

Primary Users:
Children (5–10 years)

Customers:

# Core Philosophy

لا يوجد تعقيد.

# Game Philosophy

Every game should:

Start immediately.

Be understood within seconds.

Last less than two minutes.

Encourage replay.

Require no tutorial.

Feel rewarding.

---

# MVP

The first version contains ONE game only.

Fruit Catch Game.

Nothing else.

---

# Current Game

Objects fall from the top.

Player controls a basket.

Catch fruit.

Each fruit = +1.

Missing fruit removes one heart.

Player starts with three hearts.

When hearts reach zero:

Game Over.

Display:

Final Score.

Best Score.

Play Again.

Share.

---

# Sounds

# Mobile First

Phones have the highest priority.

Large buttons.

Responsive.

Fast.

---

# Things We Will NOT Build Now

No login.

No accounts.

No backend.

No database.

No AI.

No admin panel.

No payment.

No advertisements.

No multiple games.

No settings.

No notifications.

No analytics dashboard.

---

# Product Rules

Build only what is needed today.

Do not predict future requirements.

Do not add features that were not requested.

Keep everything simple.

---

# Family Principles

Parents are customers.

Children are players.

The game should create positive family moments.

The game should never encourage harmful competition inside the family.

Progress should focus on self-improvement.

---

# User Experience

The player should smile.

The parent should trust the product.

The interface should disappear.

The experience should feel effortless.

---

# Success Criteria

A child understands the game without explanation.

A child wants to play again.

A parent is comfortable sharing it.

The game loads quickly.

The game feels polished.

---

# Long-term Direction

قشطة is not one game.

قشطة is a growing collection of short family-friendly games.

Every new game must follow the same philosophy and visual identity.

---

# Golden Rule

Whenever there is a choice between:

Adding a new feature

OR

Making the current game more enjoyable

Always improve the current game.
